# Nexus AI v1 — Foundation Release

## Introduction
Nexus AI v1 is the initial prototype of the Nexus AI project, focused on building a simple conversational AI assistant using Python and the Groq API.

This version represents the starting point of the Nexus AI development journey and demonstrates the core implementation of AI-powered text interaction.

---

## Core Features

- AI-powered chatbot
- Text-based conversation system
- Groq API integration
- Lightweight Python architecture
- Beginner-friendly implementation

## Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/maheetharan/Nexus-ai.git
```

### 2. Navigate to Version 1

```bash
cd version1
```

### 3. Install Dependencies

```bash
pip install groq
```

### 4. Run the Assistant

```bash
python Nexus.py
```
